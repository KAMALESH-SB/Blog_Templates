Train Ticket Booking System

Project Overview

This Train Ticket Booking system is a mini-project built to demonstrate core functionalities such as booking, canceling, and checking seat availability for trains. It helps users seamlessly manage their train bookings. The project uses Java with an object-oriented approach to handle train operations and ensures an efficient booking process.

Features

  • Check seat availability
  • Book tickets
  • Cancel bookings
  • Display fare details
